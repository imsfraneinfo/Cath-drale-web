# Imsfrane Cathédrale — corrected website

This package is ready to upload to the web root.

1. Copy your existing image files into `images/` using the exact names listed in `images/IMAGES-REQUIRED.txt`.
2. Upload all files and folders, including `.htaccess`, `en`, `fr`, `css`, `js`, `robots.txt`, `sitemap.xml`, and `ads.txt`.
3. In Google Search Console, submit `https://imsfrane.com/sitemap.xml`.
4. Test the booking and contact forms on a phone; they open WhatsApp with a prepared message.
